A web-based visual programming environment for exploring 3D geometry through
block-based coding. Blockly's drag-and-drop interface drives a Three.js
scene, with blocks covering vector operations, line and plane forms, and
other linear algebra concepts, so learners build visualizations by snapping
together concept-blocks instead of writing formulas. The subject of my
Honors dissertation, currently heading into a user study.
